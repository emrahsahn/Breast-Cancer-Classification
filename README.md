# Breast Cancer Classification

Bu proje, meme kanseri teşhisi için makine öğrenimi yöntemleri kullanarak veri analizi ve sınıflandırma işlemlerini gerçekleştirmektedir.

## Proje Yapısı
- `src/`: Uygulama kodları
  - `app.py`: Ana uygulama dosyası
  - `backbone.py`: Model ve yardımcı fonksiyonlar
- `data/`: Veri setleri
  - `archive (1)/data.csv`: Kullanılan veri seti
- `test/`: Test dosyaları
- `doc/`: Dokümantasyon
- `requirement.txt`: Gerekli Python paketleri

## Kurulum
1. Gerekli paketleri yükleyin:
   ```powershell
   pip install -r requirement.txt
   ```
2. Veri dosyasının `data/archive (1)/data.csv` yolunda olduğundan emin olun.

## Kullanım
Ana uygulamayı çalıştırmak için:
```powershell
python src/app.py
```
